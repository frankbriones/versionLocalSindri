*******************
**Forms clt**
*******************

Dentro de la carpeta **clt** tenemos el archivo llamado ``forms.py``. Donde se encuentran los formularios de la aplicación clt, se detalla a continuación cada formulario.

Formulario ClientesForm
========================

Hereda de los formularios de Django ``forms.ModelForm``, este formulario afecta al :ref:`Modelo Clientes`. Contiene los siguientes campos:

- identificacion
- nombres
- apellidos
- ciudad
- direccion
- telefono

- correo
