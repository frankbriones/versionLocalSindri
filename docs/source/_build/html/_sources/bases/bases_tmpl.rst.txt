*******************
**Templates Bases**
*******************

Dentro de la carpeta **bases** tenemos otra carpeta llamada **templates**. Donde se encuentran los templates de la aplicación bases

Template error404
=================

Template de la página que se mostrará cuando el usuario intenta acceder a una url errónea

Template home
=============

Template de la página principal del sistema, esta se
mostrará despues de iniciar sesión

Template login
==============

Template de la pantalla para acceder al sistema, donde se solicitará al usuario su username o correo electrónico y su contraseña para acceder

Template logout
===============

Template del modal que aparecerá cuando el usuario
requiera terminar la sesión activa

Template sin_premisos
=====================

Template de la pantalla que aparecerá cuando
el usuario intente ingresar a una parte del 
sistema de la cual no posee permisos de acceso
